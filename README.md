# 炫酷动效网站

这是一个展示现代网页设计和动画效果的示例网站。该项目使用了多种前端技术来创建引人入胜的用户体验。

## 特点

- 响应式设计，适配所有设备尺寸
- 流畅的页面过渡效果
- 3D视差效果
- 鼠标交互动画
- 滚动触发动画

## 技术栈

- HTML5
- CSS3 (包括动画、Flexbox和Grid布局)
- JavaScript (ES6+)
- GSAP (GreenSock Animation Platform)
- Three.js (3D效果)

## 如何运行

只需在浏览器中打开`index.html`文件即可查看网站。

## 浏览器兼容性

- Chrome (推荐)
- Firefox
- Safari
- Edge
